4 · Collaboration Session (Sprint / Challenge / Build)
Session Start & IDE
  • Both users join challenge room via socket
  • Project idea suggested by matching algorithm
  • Collaborative code editor (Monaco/CodeMirror)
  • Real-time file sync: create / edit / delete / rename
  • File-lock system — only 1 user edits at a time
  • Multiple terminal instances (shared output)
  • Live preview URL shared between users
  • Heartbeat/activity tracking (30s intervals)

6 · Projects Page
  • Reads only explicitly saved projects (localStorage: saved_projects)
  • Shows project title, partner name, mode, date, task progress
  • File count badge when IDE files were captured
  • Download → JSZip with all project files (or README.md fallback)
  • Delete project from list
  • Submission link → opens in new tab
  • Empty state with "Find a Partner" CTA
  • Accessible from dashboard Projects stat click
7 · Profile Page
Own Profile (/profile)
  • Edit name, bio, skills, interests, experience level
  • Upload profile photo
  • Connect / disconnect GitHub account
  • View badges earned
  • View credits & reputation
  • Completed projects count
  • Theme toggle (light/dark)
Other User Profile (/users/:userId)
  • View user's public info (name, bio, skills, badges)
  • View reputation, joined date
  • Send friend request (if not already friends)
  • Send DM via messages
  • Accessible from: friends list, collaboration session partner name, proposal/challenge requests

10 · Global Notification System
Friend Request

  • 8-second auto-dismiss
  • Accept → instant friendship

  • Green toast "🎉 You are now friends"
  • Auto-dismisses after 8s

  • Auto-dismisses after 8s

All Possible Session Outcomes
  • Normal end: Both users submit → credits awarded → projects saved
  • Time-up: Modal shown → continue or end without submitting
  • Request leave accepted: Both navigate to dashboard
  • Force quit: Partner gets credit notification + solo mode option
  • Solo mode: Continue alone with only "Leave" option
  • Partner rejoin: Session resumes seamlessly
  • Activity timeout: Slide-to-verify shown after 10min idle
  • Content violation: Warning banner in chat
  • Project rename: Accepted/declined with notification banner
